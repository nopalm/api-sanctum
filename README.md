Laravel API Sanctum

## Setup
The first thing to do is to clone the repository:

Step By Step:
```
composer install
```
Set Database:
```
setting your .env file
```
Then :
```
php artisan migrate
php artisan key:generate
```
Url:
```
http://127.0.0.1:8000/api/login
http://127.0.0.1:8000/api/products
```
